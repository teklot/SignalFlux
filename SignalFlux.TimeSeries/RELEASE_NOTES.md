# SignalFlux.TimeSeries v0.1.0

- Resample — change signal frequency with interpolation
- Interpolate — linear interpolation between samples
- Align — align multiple signals to common timestamps
- Window — extract time windows from signals
- Statistics — rolling/aggregate stats (mean, min, max, stddev, count)
- Downsample — decimate or aggregate over windows
- All operations as extension methods on Signal&lt;T&gt;
- Depends on SignalFlux.Core
- Targets netstandard2.0 + net10.0
